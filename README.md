# Space Game

A simple 2D Space Ship game built with Python and Pygame. It is a game which U should keep a live from those metroites coming to U in the wide space inspired my childhood from game * **Asteroids Arcade Console** .

## Controls

### Player 1 (Ninja):
* **FORWARD:** `W` 
* **BACKWARD:** `S`
* **RIGHT** `D`
* **LEFT:** `A`
* **Laser:** `SPACEBAR`
